# AlignADV project page

This folder contains a static GitHub Pages website for the TRC/AlignADV work.

## Files

- `index.html`: main project page.
- `assets/css/style.css`: page styling.
- `assets/images/`: figures converted from the PDFs in `TRC.zip`.

## Video placeholder

The page includes a 16:9 placeholder in the hero area, directly below the affiliations. After the project video is finalized, replace the `.hero-video` block in `index.html` with your video embed, for example an `<iframe>` or local `<video>` element.

## Deploy on GitHub Pages

1. Create a repository, for example `AlignADV` or `<username>.github.io`.
2. Copy all files in this folder to the repository root.
3. Commit and push.
4. In GitHub: **Settings → Pages → Build and deployment → Deploy from a branch**.
5. Select the branch and root folder, then save.

No build step is required.
